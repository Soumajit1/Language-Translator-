# 🌐 Language Translator Application

A simple, fast, and user-friendly **Language Translator** that allows users to translate text from one language to another using modern web technologies.

---

Overview

This project is designed to help users easily translate text between different languages in real time. It provides a clean UI, smooth user experience, and accurate translations using a translation API.


Features

- 🌍 Supports multiple languages  
- ⚡ Real-time text translation  
- 🖥️ Simple and intuitive interface  
- 🔄 Easy language switching  
- 📱 Responsive design (Desktop & Mobile)



## 🛠️ Technologies Used

- **HTML** – Structure of the application  
- **CSS** – Styling and responsiveness  
- **JavaScript** – Logic and API handling  
- **Translation API** – Language conversion
